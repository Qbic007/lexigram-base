# Техническое задание: Telegram бот для изучения иностранных языков

## 1. Общее описание проекта
### 1.1 Назначение
Telegram бот для систематического изучения иностранных языков через регулярное тестирование знаний слов и фраз. Бот отправляет пользователю слова/фразы на одном языке, а пользователь должен правильно написать их перевод на другом языке.

### 1.2 Целевая аудитория
- Люди, изучающие иностранные языки
- Пользователи, желающие систематически пополнять словарный запас
- Те, кто предпочитает учиться в удобное время через мессенджер

### 1.3 Основные преимущества
- Регулярные тренировки в удобное время
- Простой и понятный формат обучения
- Возможность настройки частоты и времени отправки заданий
- Мгновенная проверка правильности ответов

## 2. Функциональные требования
### 2.1 Основные функции
- Автоматическая отправка заданий (слов/фраз) в заданное время
- Настройка частоты отправки заданий
- Проверка правильности ответов пользователя
- Система уведомлений о новых заданиях
- Интеграция с API переводчика для мгновенного перевода
- Возможность добавления новых слов/фраз в личный словарь

### 2.2 Пользовательские сценарии
1. Настройка расписания:
   - Пользователь указывает желаемое время получения заданий
   - Пользователь выбирает частоту получения заданий (например, 2 раза в день, каждый час и т.д.)

2. Процесс обучения:
   - Бот отправляет слово/фразу на одном языке
   - Пользователь отправляет свой вариант перевода
   - Бот проверяет правильность ответа
   - Бот отправляет следующее задание согласно расписанию

3. Добавление новых слов:
   - Пользователь отправляет незнакомое слово/фразу боту
   - Бот обращается к API переводчика
   - Бот отправляет пользователю перевод
   - Бот предлагает добавить слово/фразу в личный словарь
   - При согласии пользователя слово/фразу добавляется в словарь для дальнейшего изучения

## 3. Требования к интерфейсу
### 3.1 Команды бота
- /start - начало работы с ботом
- /help - справка по использованию
- /settings - настройки расписания и частоты заданий
- /add - добавление нового слова/фразы вручную
- /stats - просмотр статистики обучения

### 3.2 Форматы сообщений
- Запрос перевода: любой текст, отправленный боту
- Ответ на задание: текстовое сообщение
- Подтверждение добавления в словарь: кнопки "Да"/"Нет"
